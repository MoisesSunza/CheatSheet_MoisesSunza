<Glosario De Etiquetas HTML 5>

¿Qué es HTML 5?

HTML5 es la quinta versión del Lenguaje de Marcado de Hipertexto (HTML), el estándar fundamental para la creación de páginas web. Esta versión no solo mantiene la función principal de estructurar el contenido (textos, imágenes, enlaces, listas, tablas, etc.), sino que también incorpora nuevas etiquetas y funcionalidades diseñadas para mejorar la semántica, la accesibilidad y la integración de contenido multimedia. Con HTML5 es posible incluir de manera nativa audio, video, gráficos y animaciones, sin depender de complementos externos como Flash. Además, introduce etiquetas semánticas como <header>, <article>, <section> y <footer>, que permiten que los documentos tengan una estructura más clara tanto para los desarrolladores como para los navegadores y motores de búsqueda. Otra de sus ventajas es que está pensado para el uso en dispositivos móviles, ofreciendo un mejor rendimiento, soporte para aplicaciones web y compatibilidad con tecnologías modernas como CSS3 y JavaScript. Gracias a estas características, HTML5 se ha convertido en la base de las aplicaciones y sitios web actuales, asegurando una experiencia más rica, interactiva y accesible para los usuarios.

Hay una gran cantidad de etiquetas que podemos utilizar, por lo que se ha dividido en secciones basadas en su función.

<Estructura>

Se refiere a las etiquetas que definen la base de una página HTML. Son las encargadas de indicar dónde comienza y termina el documento, qué parte contiene la información principal y cómo se organiza en general. Incluyen elementos como <!DOCTYPE html>, <html>, <head>, <title> y <body>. Gracias a ellas, los navegadores pueden interpretar correctamente el contenido.

<Semántica>

Agrupa etiquetas que dan un significado claro al contenido. En lugar de usar solo <div>, HTML5 incorpora elementos como <header>, <footer>, <section>, <article> o <aside>, que indican la función de cada bloque. Esto mejora la accesibilidad, el posicionamiento en buscadores (SEO) y facilita el trabajo de desarrolladores y lectores de pantalla.

Ver etiquetas de Semántica

<Contenido>

Incluye las etiquetas encargadas de mostrar información al usuario: textos, enlaces, imágenes, párrafos, títulos y citas. Ejemplos: <p>, <h1>…<h6>, <a>, <img>, <span>. Son las más utilizadas en cualquier sitio web, ya que permiten expresar información de forma clara y organizada.

Ver etiquetas de Contenido

<Listas>

Son elementos diseñados para agrupar contenido en forma de lista, ya sea numerada, con viñetas o de definiciones. Existen principalmente tres tipos: listas ordenadas <ol>, listas desordenadas <ul> y listas de definiciones <dl>. Se usan para menús, pasos de instrucciones, catálogos y más.

Ver etiquetas de Listas

<Formularios>

Engloban todas las etiquetas que permiten al usuario introducir y enviar datos. Incluyen <form>, <input>, <textarea>, <button>, <select>, entre otros. Gracias a ellos se pueden crear buscadores, registros, encuestas y cualquier tipo de interacción que requiera recopilar información.

Ver etiquetas de Formularios

<Multimedia>

Son etiquetas que integran elementos visuales y sonoros de forma nativa, sin necesidad de plugins externos. Ejemplos: <img>, <audio>, <video>. Con ellas se pueden añadir imágenes, videos, música, animaciones y gráficos interactivos, enriqueciendo la experiencia del usuario.

Ver etiquetas de Multimedia

Atributos

No son etiquetas en sí, sino propiedades que se añaden a estas para modificar su comportamiento o darles más información. Algunos son globales (como id, class, style, title) y se pueden aplicar a cualquier etiqueta, mientras que otros son específicos (como src en <img> o href en <a>). Los atributos permiten personalizar y controlar el funcionamiento de cada elemento en el documento.
