<Estructura>

Etiquetas de Estructura

La categoría de estructura agrupa las etiquetas fundamentales que definen la base de un documento HTML. Son las encargadas de establecer dónde inicia y termina la página, qué se coloca en la cabecera y qué se despliega en el cuerpo visible del navegador. Gracias a ellas, los navegadores pueden interpretar correctamente el contenido y los motores de búsqueda identificar la organización del documento. Sin estas etiquetas, una página web no podría funcionar correctamente, ya que controlan desde el tipo de documento (<!DOCTYPE html>) hasta el área visible para el usuario (<body>).

<!DOCTYPE html>

Nombre: Declaración del tipo de documento. Indica al navegador qué versión de HTML se está utilizando.

  • Características:
    • Se coloca al inicio del documento.
    • No tiene cierre.
  • Atributos: Ninguno.

Ejemplo:

<!DOCTYPE html>

<html>

Nombre: Raíz del documento HTML. Contiene todo el contenido del documento, incluyendo <head> y <body>.

  • Características:
    • Solo puede haber un <html> por documento.
    • Envuelve todas las etiquetas visibles y metadatos.
  • Atributos: Globales como lang, dir, id, class, style, data-*.

Ejemplo:

<html lang="es">
    <head>
        <title>Mi primera página</title>
    </head>
    <body>
        <p>Hola mundo</p>
    </body>
</html>

<head>

Nombre: Cabecera del documento. Contiene metadatos, enlaces a estilos, scripts, favicons y metadatos SEO.

  • Características:
    • Se coloca después de <html> y antes de <body>.
    • Solo puede haber un <head> por documento.
    • No debe contener contenido visible.
  • Atributos: Globales (id, class, style, data-*).

Ejemplo:

<head>
    <meta charset="UTF-8">
    <title>Ejemplo Head</title>
</head>

<title>

Nombre: Título del documento. Texto que aparece en la pestaña del navegador, importante para SEO.

  • Características:
    • Debe ir dentro de <head>.
    • Solo puede haber un <title> por documento.
    • No se muestra en la página.
  • Atributos: Globales (id, class, style, data-*).

Ejemplo:

<title>Página de Ejemplo</title>

<link>

Nombre: Enlace a recursos externos. Se utiliza principalmente para vincular hojas de estilo CSS, iconos o fuentes externas.

  • Características:
    • Debe colocarse dentro de <head>.
    • Es un elemento vacío (no tiene cierre).
    • Se utiliza para importar estilos, favicons, fuentes u otros recursos.
  • Atributos: Globales (id, class, style, data-*), href, rel, type, media.

Ejemplo:

<head>
    <link rel="stylesheet" href="estilos.css">
    <link rel="icon" href="favicon.ico" type="image/x-icon">
</head>

<meta>

Nombre: Metadato del documento. Proporciona información adicional sobre la página web que no se muestra directamente al usuario.

  • Características:
    • Debe colocarse dentro de <head>.
    • Es un elemento vacío (no tiene cierre).
    • Se utiliza para definir la codificación, autor, descripción, palabras clave, viewport y SEO.
  • Atributos: Globales (id, class, style, data-*), charset, name, content, http-equiv.

Ejemplo:

<head>
    <meta charset="UTF-8">
    <meta name="description" content="Ejemplo de uso de meta">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

<body>

Nombre: Cuerpo del documento. Contiene todo el contenido visible para el usuario como textos, imágenes, enlaces, listas y multimedia.

  • Características:
    • Solo puede haber un <body> por documento.
    • Se coloca después de <head>.
  • Atributos: Globales y específicos como background, onload, onunload.

Ejemplo:

<body>
    <h1>Contenido visible</h1>
    <p>Este texto lo ve el usuario en el navegador.</p>
</body>
