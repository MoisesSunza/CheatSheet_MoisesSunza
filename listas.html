<Listas>

Etiquetas de Listas

Las listas son un recurso esencial para organizar información de manera jerárquica o secuencial. HTML permite tres tipos principales: listas ordenadas (<ol>) con numeración automática, listas desordenadas (<ul>) con viñetas, y listas de definición (<dl>) que sirven para glosarios o pares de término-definición. Además, estas etiquetas se combinan con <li> (elementos de lista) o <dt> y <dd> en el caso de las listas de definición. Son muy usadas en menús, pasos de instrucciones, catálogos y documentos que necesitan una organización clara.

<ol>

Representa una lista ordenada, normalmente numerada.

  • Características:
    • Se usa para pasos o secuencias.
    • Contiene elementos <li>.
  • Atributos: Globales, type, start, reversed.

Ejemplo:

<ol type="1" start="3">
    <li>Paso uno</li>
    <li>Paso dos</li>
    <li>Paso tres</li>
</ol>

Resultado:

  1. Paso uno
  2. Paso dos
  3. Paso tres

<ul>

Representa una lista desordenada, normalmente con viñetas.

  • Características:
    • Se usa para agrupar elementos sin orden.
    • Contiene elementos <li>.
  • Atributos: Globales.

Ejemplo:

<ul>
    <li>Manzana</li>
    <li>Banana</li>
    <li>Naranja</li>
</ul>

Resultado:

  • Manzana
  • Banana
  • Naranja

<li>

Representa un elemento dentro de una lista ordenada (<ol>) o desordenada (<ul>).

  • Características:
    • Contiene texto o incluso otras listas.
  • Atributos: Globales, value (para <ol>).

Ejemplo:

<ol>
    <li value="10">Elemento con valor 10</li>
    <li>Elemento normal</li>
</ol>

Resultado:

  1. Elemento con valor 10
  2. Elemento normal

<dl>

Representa una lista de definiciones, donde se relacionan términos con descripciones.

  • Características:
    • Contiene pares de <dt> y <dd>.
  • Atributos: Globales.

Ejemplo:

<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para estructurar páginas web.</dd>
    <dt>CSS</dt>
    <dd>Lenguaje de estilos para dar diseño a páginas web.</dd>
</dl>

Resultado:

HTML
Lenguaje de marcado para estructurar páginas web.
CSS
Lenguaje de estilos para dar diseño a páginas web.

<dt>

Representa el término dentro de una lista de definiciones (<dl>).

  • Características:
    • Se combina con <dd> para definir la descripción.
  • Atributos: Globales.

Ejemplo:

<dl>
    <dt>CPU</dt>
    <dd>Unidad central de procesamiento, el cerebro del computador.</dd>
</dl>

Resultado:

CPU
Unidad central de procesamiento, el cerebro del computador.

<dd>

Representa la descripción de un término dentro de una lista de definiciones (<dl>).

  • Características:
    • Debe seguir a un <dt>.
  • Atributos: Globales.

Ejemplo:

<dl>
    <dt>RAM</dt>
    <dd>Memoria de acceso aleatorio, usada para almacenar datos temporales.</dd>
</dl>

Resultado:

RAM
Memoria de acceso aleatorio, usada para almacenar datos temporales.
