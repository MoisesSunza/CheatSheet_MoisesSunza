Atributos

Atributos de etiquetas

Los atributos no son etiquetas en sí, sino propiedades que complementan a estas para modificar su comportamiento, darles significado extra o personalizarlas. Algunos son globales y se aplican a casi cualquier elemento, como id, class o style. Otros son específicos de ciertas etiquetas, como src en <img>, href en <a> o action en <form>. Gracias a los atributos, se pueden definir enlaces, rutas de recursos, tamaños, estilos visuales, validaciones de formularios y mucho más. Son una parte clave de HTML porque permiten adaptar cada etiqueta a las necesidades concretas del sitio.

Atributos Globales

Pueden aplicarse a casi cualquier elemento HTML y modifican su comportamiento o apariencia.

  • id: Identificador único del elemento.
  • class: Clase(s) para aplicar estilos o seleccionar elementos con CSS/JS.
  • style: Estilos CSS en línea.
  • title: Texto que aparece al pasar el mouse sobre el elemento.
  • data-*: Almacena datos personalizados accesibles mediante JavaScript.
  • hidden: Oculta el elemento de la vista del usuario.
  • accesskey: Tecla rápida para enfocar o activar el elemento.
  • tabindex: Define el orden de tabulación del elemento.

Define el idioma principal del contenido del documento.

head, title, meta, link

  • charset (meta): Define la codificación de caracteres (UTF-8, ISO-8859-1, etc.).
  • name (meta): Nombre del metadato, por ejemplo description o viewport.
  • content (meta): Valor del metadato.
  • http-equiv (meta): Proporciona encabezados HTTP equivalentes.
  • viewport (meta): Configura el área visible de la página en dispositivos móviles.
  • rel (link): Relación del enlace con el documento (stylesheet, icon, etc.).
  • href (link): URL del recurso vinculado.
  • type (link): Tipo de recurso vinculado (text/css, etc.).
  • media (link): Define el tipo de dispositivo (screen, print, etc.).
  • crossorigin (link): Controla el envío de credenciales para recursos externos.
  • integrity (link): Verifica la integridad de un recurso externo.

Atributos de tablas

  • border (table): Define grosor del borde (obsoleto, se recomienda CSS).
  • summary (table): Descripción de la tabla para lectores de pantalla.
  • width (table): Ancho de la tabla (recomendado CSS).
  • colspan (td, th): Número de columnas que ocupa la celda.
  • rowspan (td, th): Número de filas que ocupa la celda.
  • headers (td): Relaciona una celda con cabeceras (id de th).
  • scope (th): Define el alcance de la celda de cabecera (col, row, colgroup, rowgroup).

Atributos de listas

  • type (ol): Tipo de numeración (1, a, i, etc.).
  • start (ol): Número inicial de la lista.
  • reversed (ol): Lista en orden descendente.
  • value (li): Valor numérico específico para un ítem de lista.

Atributos de Formularios

  • action (form): URL a la que se envían los datos del formulario.
  • method (form): Método de envío (GET o POST).
  • enctype (form): Tipo de codificación de los datos enviados (multipart/form-data, application/x-www-form-urlencoded, text/plain).
  • autocomplete (form): Activa o desactiva la autocompletación del formulario.
  • novalidate (form): Indica que no se valide el formulario al enviarlo.
  • target (form): Ventana o marco donde se abre el resultado.

Atributos de <input>

  • type: Tipo de control (text, password, radio, checkbox, submit, file, etc.).
  • name: Nombre del campo.
  • value: Valor inicial o enviado del campo.
  • placeholder: Texto de ayuda dentro del campo.
  • required: Indica que el campo es obligatorio.
  • readonly: Campo de solo lectura.
  • disabled: Desactiva el campo.
  • maxlength, minlength: Número máximo/mínimo de caracteres.
  • autofocus: Enfoca automáticamente al cargar la página.

Atributos de <textarea>

  • name: Nombre del campo.
  • rows: Número de filas visibles.
  • cols: Número de columnas visibles.
  • placeholder: Texto de ayuda dentro del área de texto.
  • required, readonly, disabled, maxlength, minlength: Igual que <input>.

Atributos de <button>

  • type: submit, reset, button.
  • disabled: Desactiva el botón.
  • name, value: Para identificar el botón en formularios.
  • autofocus: Enfoca automáticamente al cargar la página.
  • form: Asocia el botón a un formulario específico.

Atributos de <label>

  • for: Id del control de formulario asociado.
  • form: Asocia la etiqueta a un formulario específico.

Atributos de <select> y <option>

  • <select>: name, size, multiple, required, disabled.
  • <option>: value, selected, disabled, label.

Atributos de <fieldset> y <legend>

  • <fieldset>: disabled, form, name.
  • <legend>: accesskey, autofocus, form, hidden.

Atributos de <img>

  • src: URL de la imagen.
  • alt: Texto alternativo.
  • width, height: Dimensiones.
  • crossorigin: Control de credenciales para imágenes externas.
  • usemap: Mapa de imagen.
  • longdesc: URL de descripción detallada.
  • loading: lazy o eager (carga diferida).
  • referrerpolicy: Política de referencia HTTP.

Atributos de <audio>, <video> y <source>

  • <audio>: src, controls, autoplay, loop, muted, preload.
  • <video>: src, controls, autoplay, loop, muted, preload, poster, width, height.
  • <source>: src, type, media.
