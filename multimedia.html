<Multimedia>

Etiquetas de Multimedia

La categoría de multimedia abarca etiquetas que enriquecen la experiencia del usuario integrando imágenes, audio y video de manera nativa, sin necesidad de complementos externos como Flash. Gracias a ellas, una página puede mostrar contenido más interactivo y atractivo, como música (<audio>), películas o animaciones (<video>), podcasts o galerías de fotos (<img>). Estas etiquetas también incluyen atributos para controlar la reproducción automática, la carga diferida, los subtítulos y la accesibilidad, permitiendo un uso más inclusivo y adaptable en distintos dispositivos, desde computadoras hasta móviles.

<img>

Representa una imagen en la página web.

  • Características:
    • Elemento vacío (sin cierre).
    • Se usa para mostrar imágenes estáticas en diversos formatos.
  • Atributos: Globales, src, alt, width, height, crossorigin, usemap, longdesc, loading, referrerpolicy.

Ejemplo:

<img src="imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

<audio>

Permite incrustar contenido de audio reproducible en la página.

  • Características:
    • Se puede controlar con atributos como controls, autoplay, loop y muted.
    • Puede contener uno o más elementos <source> para distintos formatos.
  • Atributos: Globales, src, controls, autoplay, loop, muted, preload.

Ejemplo:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Tu navegador no soporta audio HTML5.
</audio>

Resultado:

<video>

Permite incrustar contenido de video reproducible en la página.

  • Características:
    • Se puede controlar con atributos como controls, autoplay, loop, muted y poster.
    • Puede contener uno o más elementos <source> para distintos formatos de video.
  • Atributos: Globales, src, controls, autoplay, loop, muted, preload, poster, width, height.

Ejemplo:

<video width="320" height="240" controls poster="poster.jpg">
    <source src="video.mp4" type="video/mp4">
    Tu navegador no soporta video HTML5.
</video>

Resultado:

<source>

Define un recurso multimedia alternativo para elementos de audio o video.

  • Características:
    • Se usa dentro de <audio> o <video> para indicar diferentes formatos.
    • No se utiliza solo; siempre debe estar dentro de un contenedor multimedia.
  • Atributos: Globales, src, type, media.

Ejemplo:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
    <source src="video.mp4" type="video/mp4">
</video>
