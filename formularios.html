Etiquetas de Formularios

Las etiquetas de formularios son las que permiten la interacción entre el usuario y la página web. Con ellas, se pueden introducir datos, hacer búsquedas, registrarse, enviar comentarios o cargar archivos. HTML ofrece gran variedad de controles como campos de texto (<input>), menús desplegables (<select>), botones (<button>), áreas de texto (<textarea>), casillas de verificación y más. Los formularios son esenciales para sitios dinámicos, ya que permiten recolectar información y enviarla al servidor, donde puede ser procesada para generar respuestas personalizadas o almacenar datos en bases.

<form>

Representa un formulario HTML que permite a los usuarios enviar datos al servidor.

  • Características:
    • Contiene controles como <input>, <textarea>, <select> y botones.
    • Se puede enviar mediante métodos GET o POST.
  • Atributos: Globales, action, method, enctype, autocomplete, novalidate, target.

Ejemplo:

<form action="/procesar.php" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
    <button type="submit">Enviar</button>
</form>

<input>

Representa un control interactivo para introducir datos.

  • Características:
    • Puede ser de tipo texto, contraseña, radio, checkbox, file, submit, entre otros.
    • No tiene etiqueta de cierre.
  • Atributos: Globales, type, name, value, placeholder, required, readonly, disabled, maxlength, minlength.

Ejemplo:

<input type="text" name="usuario" placeholder="Nombre de usuario">
<input type="password" name="clave" placeholder="Contraseña">
<input type="checkbox" name="suscribir"> Suscribirse
<input type="radio" name="genero" value="M"> Masculino
<input type="radio" name="genero" value="F"> Femenino
<input type="file" name="archivo">
<input type="submit" value="Enviar">

Resultado:

Suscribirse Masculino Femenino

<select>

Representa un menú desplegable que permite seleccionar una o varias opciones.

  • Características:
    • Contiene elementos <option>.
    • Puede permitir selección múltiple usando el atributo multiple.
  • Atributos: Globales, name, size, multiple, required, disabled.

<option>

Representa una opción dentro de un <select>.

  • Características:
    • Se puede seleccionar individualmente o formar parte de un grupo de opciones.
    • Se puede marcar como seleccionada por defecto con selected.
  • Atributos: Globales, value, selected, disabled, label.

Ejemplo <select> y <option>:

<label for="pais">País:</label>
<select id="pais" name="pais">
    <option value="mx" selected>México</option>
    <option value="ar">Argentina</option>
    <option value="es">España</option>
</select>

Resultado:

<textarea>

Representa un campo de texto multilinea donde el usuario puede escribir varias líneas.

  • Características:
    • Se puede controlar tamaño con rows y cols.
    • Se puede establecer valor inicial dentro de la etiqueta.
  • Atributos: Globales, name, rows, cols, placeholder, required, readonly, disabled, maxlength, minlength.

Ejemplo:

<label for="comentario">Comentario:</label>
<textarea id="comentario" name="comentario" rows="4" cols="40" placeholder="Escribe tu mensaje aquí"></textarea>

Resultado:

<label>

Representa la etiqueta de un control de formulario, mejorando accesibilidad y usabilidad.

  • Características:
    • Se puede asociar a un control mediante el atributo for.
    • Mejora la interacción para lectores de pantalla y clics de usuario.
  • Atributos: Globales, for, form.

Ejemplo:

<label for="correo">Correo electrónico:</label>
<input type="email" id="correo" name="correo" required>

Resultado:

<button>

Representa un botón interactivo que puede enviar un formulario, resetearlo o ejecutar código.

  • Características:
    • Se puede usar dentro o fuera de un formulario.
    • Puede contener texto, imágenes o HTML en su interior.
  • Atributos: Globales, type (submit, reset, button), disabled, name, value, autofocus, form.

Ejemplo:

<button type="submit">Enviar</button>
<button type="reset">Reiniciar</button>
<button type="button" onclick="alert('Hola!')">Saludar</button>

Resultado:

<fieldset>

Agrupa controles de un formulario de manera lógica y semántica.

  • Características:
    • Se suele usar con <legend> para describir el grupo.
    • Mejora accesibilidad y organización visual del formulario.
  • Atributos: Globales, disabled, form, name.

<legend>

Representa un título o descripción para un grupo de controles dentro de <fieldset>.

  • Características:
    • Debe colocarse como primer hijo dentro de <fieldset>.
    • Ayuda a identificar el propósito del grupo de campos.
  • Atributos: Globales, accesskey, autofocus, form, hidden.

Ejemplo <fieldset> y <legend>:

<fieldset>
    <legend>Datos Personales</legend>
    <label for="nombre2">Nombre:</label>
    <input type="text" id="nombre2" name="nombre2">
    <label for="edad">Edad:</label>
    <input type="number" id="edad" name="edad">
</fieldset>

Resultado:

Datos Personales
